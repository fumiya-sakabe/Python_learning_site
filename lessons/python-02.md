# 最初の一歩：printと計算

## このレッスンのゴール

- `print()`関数を使って画面に文字を表示できる
- Pythonで基本的な計算（四則演算）ができる
- コメントを書いてコードを読みやすくできる
- エラーメッセージを読んで、簡単な修正ができる

## なぜ重要なのか

`print()`は、プログラミングの最初の一歩として、最も基本的で重要な関数です。

デバッグや動作確認の際にも頻繁に使われます。また、計算機能はプログラミングの基礎であり、
実際のアプリケーションでも数値計算は欠かせません。

このレッスンで基本的な出力と計算をマスターすることで、Pythonの基本操作に慣れることができます。

## 解説

このレッスンでは、Pythonで最も基本的な操作を学びます。まずは「画面に何かを表示する」ことから始めましょう。プログラミングは「コンピュータに指示を出す」ことです。`print()`は「これを表示して」という指示を出すための<strong class="term">関数</strong>（機能をまとめたもの）です。

### 1. print()関数で文字を表示する

<strong class="term">関数</strong>とは、特定の処理を行うためのコードの塊です。`print()`は「引数に渡した内容を画面に表示する」という機能を持っています。

<strong class="term">引数</strong>とは、関数に渡す情報のことです。例えば、`print("Hello")`では、`"Hello"`が引数です。

まずは、文字を表示してみましょう：

```python
print("Hello, World!")
print("Pythonを学ぼう")
```

このコードを実行すると、画面に以下のように表示されます：
```
Hello, World!
Pythonを学ぼう
```

<strong class="highlight">ポイント：</strong>
- `print()`の後に`()`を付けるのを忘れないようにしましょう
- 文字列は`" "`（ダブルクォート）または`' '`（シングルクォート）で囲みます
- 日本語も表示できます

数値も表示できます：

```python
print(42)
print(3.14)
```

複数の値を一度に表示することもできます：

```python
print("答えは", 42, "です")
# 出力: 答えは 42 です
```

### 2. 基本的な計算（四則演算）

Pythonでは、電卓のように数値の計算ができます。<strong class="term">四則演算</strong>とは、足し算・引き算・掛け算・割り算のことです。

計算結果を画面に表示するには、`print()`の中に計算式を書きます。Pythonが自動で計算して、結果を表示してくれます。

```python
# 足し算（+）
print(3 + 5)  # 8

# 引き算（-）
print(10 - 4)  # 6

# 掛け算（*）：アスタリスクで表します
print(4 * 3)  # 12

# 割り算（/）：スラッシュで表します
print(10 / 3)  # 3.3333333333333335（小数点以下も表示）

# 割り算（整数除算）（//）：小数点以下を切り捨てます
print(10 // 3)  # 3

# 余り（%）：割り算の余りを求めます
print(10 % 3)  # 1（10を3で割った余り）

# べき乗（**）：累乗を求めます（2の3乗）
print(2 ** 3)  # 8（2 × 2 × 2 = 8）
```

<strong class="highlight">覚え方：</strong>
- `+`：足し算
- `-`：引き算
- `*`：掛け算（×の代わり）
- `/`：割り算（÷の代わり）
- `//`：整数の割り算（小数点以下を切り捨て）
- `%`：余りを求める
- `**`：べき乗（2の3乗など）

計算の優先順位も数学と同じです。掛け算・割り算が先に計算され、その後に足し算・引き算が計算されます：

```python
print(2 + 3 * 4)  # 14（掛け算が先）
print((2 + 3) * 4)  # 20（括弧が先）
```

### 3. 文字列と数値の結合

文字列（`"Hello"`のような文字）と数値（`25`のような数字）は、Pythonでは別の種類のデータとして扱われます。これを<strong class="term">型</strong>と言います。

文字列と数値をそのまま組み合わせようとするとエラーになります。例えば：
```python
age = 25
print("私は" + age + "歳です")  # エラー！
```

このエラーを避けるために、数値を文字列に変換する必要があります。`str()`という関数を使うと、数値を文字列に変換できます：

```python
age = 25
print("私は" + str(age) + "歳です")
# または
print(f"私は{age}歳です")  # f-string（後で詳しく学びます）
```

### 4. コメントの書き方

<strong class="term">コメント</strong>とは、コードの中に書く説明文のことです。コメントはプログラムの実行には影響せず、人間がコードを理解するためのメモとして使います。

なぜコメントが必要なのでしょうか？
- 後からコードを見た時に、何をしているか思い出しやすい
- 他の人にコードの意味を伝えやすい
- コードが長くなった時に整理しやすい

コメントの書き方は以下の通りです：

```python
# これは1行のコメントです
print("Hello")  # 行の後ろにもコメントを書けます

"""
複数行のコメントも書けます
複数行にわたって説明を書くことができます
"""
```

## よくある間違い

- 文字列に数値を直接連結しようとする（`"年齢: " + 25` → `str(25)`に変換が必要）
- 計算の優先順位を間違える（括弧を使って明確にする）
- `print`の後に括弧を忘れる（`print "Hello"` → `print("Hello")`）
- シングルクォートとダブルクォートを混同する（どちらでも良いが、統一する）
- 日本語の文字コードエラー（ファイルの文字コードをUTF-8にする）

   - 100 + 50
   - 100 - 50
   - 100 * 50
   - 100 / 50

問3. 自己紹介文を作成してください（名前、年齢、好きなことなどを表示）

```python
name = "山田太郎"
age = 25
hobby = "読書"
# print文で自己紹介を表示
```

<details>
<summary>解答を表示</summary>

```python

age = 25
hobby = "読書"
print("私の名前は", name, "です。年齢は", age, "歳、趣味は", hobby, "です。")

print(f"私の名前は{name}です。年齢は{age}歳、趣味は{hobby}です。")
```

</details>

問4. BMI（体重kg ÷ 身長m²）を計算するプログラムを作成してください。

```python
weight = 65  # kg
height = 1.70  # m
# BMIを計算して表示
```

<details>
<summary>解答を表示</summary>

```python
weight = 65
height = 1.70
bmi = weight / (height ** 2)
print(bmi)
# 小数点2桁で表示
print(f"BMI: {bmi:.2f}")
```
</details>
