# 最初の一歩：printと計算

## このレッスンのゴール

- `print()`関数を使って画面に文字を表示できる
- Pythonで基本的な計算（四則演算）ができる
- コメントを書いてコードを読みやすくできる
- エラーメッセージを読んで、簡単な修正ができる

## なぜ重要なのか

`print()`は、プログラミングの最初の一歩として、最も基本的で重要な関数です。

デバッグや動作確認の際にも頻繁に使われます。また、計算機能はプログラミングの基礎であり、
実際のアプリケーションでも数値計算は欠かせません。

このレッスンで基本的な出力と計算をマスターすることで、Pythonの基本操作に慣れることができます。

## 解説

### 1. print()関数で文字を表示する

`print()`関数は、引数に渡した内容を画面に表示します。

```python
print("Hello, World!")
print("Pythonを学ぼう")
```

数値も表示できます：

```python
print(42)
print(3.14)
```

複数の値を一度に表示することもできます：

```python
print("答えは", 42, "です")
# 出力: 答えは 42 です
```

### 2. 基本的な計算（四則演算）

Pythonでは、数値の計算が簡単にできます。

```python
# 足し算
print(3 + 5)  # 8

# 引き算
print(10 - 4)  # 6

# 掛け算
print(4 * 3)  # 12

# 割り算
print(10 / 3)  # 3.3333333333333335

# 割り算（整数除算）
print(10 // 3)  # 3

# 余り
print(10 % 3)  # 1

# べき乗
print(2 ** 3)  # 8
```

計算の優先順位も数学と同じです：

```python
print(2 + 3 * 4)  # 14（掛け算が先）
print((2 + 3) * 4)  # 20（括弧が先）
```

### 3. 文字列と数値の結合

文字列と数値を組み合わせるには、型を変換する必要があります：

```python
age = 25
print("私は" + str(age) + "歳です")
# または
print(f"私は{age}歳です")  # f-string（後で詳しく学びます）
```

### 4. コメントの書き方

コードに説明を加えるためにコメントを使います：

```python
# これは1行のコメントです
print("Hello")  # 行の後ろにもコメントを書けます

"""
複数行のコメントも書けます
複数行にわたって説明を書くことができます
"""
```

## よくある間違い

- 文字列に数値を直接連結しようとする（`"年齢: " + 25` → `str(25)`に変換が必要）
- 計算の優先順位を間違える（括弧を使って明確にする）
- `print`の後に括弧を忘れる（`print "Hello"` → `print("Hello")`）
- シングルクォートとダブルクォートを混同する（どちらでも良いが、統一する）
- 日本語の文字コードエラー（ファイルの文字コードをUTF-8にする）

## 演習課題

1. 自分の名前を`print()`で表示してください。

```python
print("私の名前は[あなたの名前]です")
```

<details>
<summary>解答を表示</summary>

```python
print("私の名前は山田太郎です")
```
</details>

2. 以下の計算を実行して結果を表示してください：
   - 100 + 50
   - 100 - 50
   - 100 * 50
   - 100 / 50
   - 100 ** 2（100の2乗）

<details>
<summary>解答を表示</summary>

```python
print(100 + 50)   # 150
print(100 - 50)   # 50
print(100 * 50)   # 5000
print(100 / 50)   # 2.0
print(100 ** 2)   # 10000
```
</details>

3. 自己紹介文を作成してください（名前、年齢、好きなことなどを表示）

```python
name = "山田太郎"
age = 25
hobby = "読書"
# print文で自己紹介を表示
```

<details>
<summary>解答を表示</summary>

```python
name = "山田太郎"
age = 25
hobby = "読書"
print("私の名前は", name, "です。年齢は", age, "歳、趣味は", hobby, "です。")
# f-string 版
print(f"私の名前は{name}です。年齢は{age}歳、趣味は{hobby}です。")
```
</details>

4. BMI（体重kg ÷ 身長m²）を計算するプログラムを作成してください。

```python
weight = 65  # kg
height = 1.70  # m
# BMIを計算して表示
```

<details>
<summary>解答を表示</summary>

```python
weight = 65
height = 1.70
bmi = weight / (height ** 2)
print(bmi)
# 小数点2桁で表示
print(f"BMI: {bmi:.2f}")
```
</details>
